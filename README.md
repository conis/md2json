md2json
=======

#Introduction

md2json可以提取带有meta的Markdown内容，并返回一个JSON格式的数据。一个很常见的应用，是提取博客文章中的meta。md2json支持自定义meta中的key，并且可以设置数据格式

#Usage

1. `npm install md2json`
2. 请参考测试用例
